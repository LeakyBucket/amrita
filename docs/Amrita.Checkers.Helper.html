defchecker(name, _ // quote() do _ end, contents)

Helper function to create your own checker functions.

Example:

defchecker thousand(actual) do
  actual |> 1000
end

fact "using thousand checker" do
  1000 |> thousand
  1001 |> ! thousand
end
