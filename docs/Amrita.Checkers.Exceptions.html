raises(function, expected_exception)

Checks if an exception was raised and that it was of the expected type or matches the expected message. Note it does not currently match when throw (Erlang errors) .

Example

fn -> raise Exception end |> raises Exception ; true
fn -> raise "Jolly jolly gosh" end |> raises %r"j(w)+y" ; true

fn -> true end            |> raises Exception ; false
